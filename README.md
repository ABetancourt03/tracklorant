## Run Locally

Clone the project

```bash
  git clone https://github.com/ABetancourt03/tracklorant.git
```

Go to the project directory

```bash
  cd tracklorant
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm start or pnpm android/ios
```
